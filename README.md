## CRUD com banco de dados MongoDB

- Certifique-se de que você tem o mongoDB rodando
- Verifique os parâmetros de segurança (Este repositótio, por padrão, conecta-se em uma sessão sem senha)

Executando:
`npm run dev`

### Rotas
- GET /users
- GET /users/:id
- POST /users
- PUT /users/:id
- DELETE /users/:id
